# alphakit-strategies-trend

Trend-following strategies for [AlphaKit](https://github.com/ankitjha67/alphakit).

Phase 0: one reference strategy (`tsmom_12_1`).
Phase 1 target: 15 strategies (TSMOM variants, cross-sectional momentum,
dual momentum, residual momentum, 52-week high, SMA/EMA crosses, Donchian
breakouts, Turtle, Ichimoku, SuperTrend).
