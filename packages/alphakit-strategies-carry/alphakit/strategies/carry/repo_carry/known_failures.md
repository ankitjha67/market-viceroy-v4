# Known failure modes — repo_carry

## 1. No actual repo rate data
Phase 1 proxy uses price Z-score, not actual GC/special repo rates.

## 2. Liquidity crises
During funding crunches, repo rates spike and specialness disappears.

## 3. Regulatory changes
Basel III and other regulations have compressed repo spreads.

## 4. Settlement risk
Repo fails can disrupt the carry trade.
