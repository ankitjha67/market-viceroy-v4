# Known failure modes — cross_asset_carry

## 1. Correlation spikes in crises
Cross-asset diversification breaks down when correlations spike.

## 2. Carry crowding
Widely implemented across asset managers, increasing crash severity.

## 3. Proxy conflation
Single trailing-return signal conflates carry with momentum.

## 4. Equal-weight assumption
True KMPV uses inverse-vol weighting per sleeve; this implementation
applies uniform cross-sectional ranking.
