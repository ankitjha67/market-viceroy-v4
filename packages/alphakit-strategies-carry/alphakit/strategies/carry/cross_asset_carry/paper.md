## Summary

KMPV define carry universally across asset classes and show that
a diversified cross-asset carry portfolio has a higher Sharpe ratio
than single-asset-class carry.

## Phase 1 proxy

Uses risk-adjusted trailing return as a uniform carry signal across
all assets. True implementation uses asset-class-specific carry
definitions with inverse-vol weighting. See ADR-001.
