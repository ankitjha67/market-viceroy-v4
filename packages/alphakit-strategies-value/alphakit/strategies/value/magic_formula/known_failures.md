# Known failure modes — magic_formula

## 1. Proxy limitation
Price-derived proxy does not capture the paper's actual signal. See docs/deviations.md.

## 2. Value trap risk
Stocks that appear cheap may be cheap for fundamental reasons (distress, declining industry).

## 3. Regime sensitivity
Value premium has been weak in some periods (e.g., 2017-2020 growth dominance).

## 4. Look-ahead bias risk
Long lookback windows reduce available backtest history and may introduce survivorship bias.
