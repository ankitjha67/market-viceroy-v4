# Book — Shareholder Yield

> Faber, M.T. (2013). ISBN 978-0-988-67950-0.

## Summary
Faber ranks stocks by total shareholder yield (dividends + buybacks + debt paydown). Phase 1 proxy: fraction of positive monthly returns. See ADR-001.

## Parameters
| Parameter | Default |
|---|---|
| Lookback | 252 days (1 year) |
