# alphakit-bridges

Engine bridges that translate a `StrategyProtocol` into backtests on:

- Internal vectorized engine (default, zero extra deps)
- vectorbt (optional, `pip install alphakit-bridges[vectorbt]`)
- backtrader (optional, `pip install alphakit-bridges[backtrader]`)
- LEAN (Phase 2+, stub)
- nautilus (Phase 4+, stub)
